# Applied AI Course Repository

## Overview

This repository contains code implementations, datasets, and materials from the lab sessions of the Applied AI course at Concordia University. It is intended for students enrolled in the course to access and review the practical components of the curriculum.

## Prerequisites

To run the code in this repository, you will need:

- Python 3.x
- Jupyter Notebook or JupyterLab
- Required Python libraries: scikit-learn, pandas, numpy, matplotlib (install via `pip install scikit-learn pandas numpy matplotlib`)

## Usage

1. Clone the repository to your local machine.
2. Navigate to the repository directory.
3. Open the Jupyter Notebook files using Jupyter Notebook or JupyterLab.
4. Run the cells in the notebook to execute the code and view the results.
